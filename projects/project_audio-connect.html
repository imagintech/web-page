About the project

Motivation

The aim of this project is to stream audio from one device to multiple other devices. One particular use case which drove this project is assuming a bunch of friends trying to watch a video on a device and the device's volume level of speakers is too low or they don't want to disturb nearby people. Using this app that audio can be streamed to multiple mobiles and people can hear(preferably via plugged-in headphones) seamlessly.

Basic Idea

Create a server, capture audio data and start streaming over the network to the connected clients using UDP protocol. On client side, receive the audio data and play via device's speakers.

Future Work

To enable streaming from any audio-connect device to multiple audio-connect devices

Installation

Server
Linux

Download the .pkg file from Downloads section and double click to install the app. Please go through the sections Configuration and Usage for further instructions.

Important Note:

  1. The app actually streams the dataline of your microphone to the mobile client, as of now it do not have the capability of detecting currently playing datalines. So you need to give permission to access the microphone.
  2. In order to listen to any audio played on your desktop, you have to rely on Operating System based software through which you can select the corresponding audio to be streamed to the app.

App
Android

Download the app from android play store and install on your mobile.

Configuration

Using PulseAudio Volume Control in Linux to stream any audio
(You need to do this only once)
  1. Make sure you have PulseAudio Volume Control(PAVU Control) installed on your system
  2. Start the audio-connect app (don't forget to give microphone access)
  3. Open PAVU Control, you will see something like below



  4. Go to the Recording section, you can see the audio-connect app and select Monitor of Built-in Audio Analog Stereo to stream from



  5. Hurray! you are done with the server setup

Usage

To stream from a computer to mobiles
  1. Make sure you have audio-connect server installed and configured on your system & audio-connect app installed on your mobile device
  2. Also, make sure all your devices are connected to same WiFi or all mobiles to hotspot of one computer
  3. Start the server app on your computer by clicking on start-server button. Wait till server status displayed as RUNNING...
  4. On your mobile, open the audio-connect app and press the search button (bottom-center) to search the for the server
  5. The detected server's IP and PORT are shown at the center
  6. Press 'connect' button (bottom-right) to listen to the stream.
  7. Enjoy the new audio experience!, email us for any help
