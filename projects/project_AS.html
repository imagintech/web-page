About the project

Motivation

The aim of this project is to stream audio from one device to multiple other devices. One particular use case which drove this project is assuming a bunch of friends trying to watch a video on a device and the device's volume level of speakers is too low or they don't want to disturb nearby people. Using this app that audio can be streamed to multiple mobiles and people can hear(preferably via plugged-in headphones) seamlessly.

Basic Idea

Create a server, capture audio data and start streaming over the network to the connected clients using UDP protocol. On client side, receive the audio data and play via device's speakers.

Future Work

To enable streaming from any audio-connect device to multiple audio-connect devices

How to use
To stream from a computer to mobiles
  • Make sure all your devices are connected to same WiFi or all mobiles to hotspot of one computer
  • Start the server app on your computer by clicking on start-server button. Wait till server status displayed as RUNNING...
  • On your mobile open the audio-connect app and press the search button (bottom-center) to search the for the server
  • The detected server's IP and PORT are shown at the center
  • Press 'connect' button (bottom-right) to listen the stream.
